【IPFS】初探IPFS

什么是IPFS

IPFS（InterPlanetary File System），中文称为星际文件系统。IPFS是一个互联网的底层协议，基于区块链的，永久的、去中心化保存和共享文件的方法，是一种P2P的分布式协议。目标是打造一个更加开放、快速、安全的互联网，利用分布式哈希表解决数据的传输和定位问题，把点对点的单点传输改变成P2P（多点对多点）的传输，其中存储数据的结构是哈希链。
